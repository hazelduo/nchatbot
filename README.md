<Raspberry Pi Commands>
-> to set gpio to output 
<Format> &output gpio-pin-number
[Example]: &output 12

-> to turn on gpio
<Format> &on gpio-pin-number
[Example]: &on 12

-> to turn off gpio
<Format> &off gpio-pin-number
[Example]: &off 12

-> For Bash Command
type $ before message
[Example]: $ls
